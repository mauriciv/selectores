Selectores

Sirven para seleccionar un elementos HTML, tanto en CSS como en Javascript

Selector universal (*)

Coincide con elementos de cualquier tipo

CSS

* {
    color: red;
}

Javascript

document.querySelectorAll('*')
Ejemplo

Selectores de tipo

Coincide con elementos de cuyo nombre coincide con lo escrito

CSS

div {
    color: red;
}

Javascript

document.querySelectorAll('div')
Ejemplo

Selectores de clase (.)

Coincide con elementos que tienen las clases especificadas

CSS

.rojo {
    color: red;
}

Javascript

document.querySelectorAll('.rojo')
Ejemplo

Selectores de id (#)

Coincide con elementos que tienen el id especificado

CSS

#unico {
    color: red;
}

Javascript

document.querySelectorAll('#unico')
Ejemplo

Selectores de atributo ([])

Coincide con elementos que tienen el atributo especificado

CSS

[href] {
    color: red;
}

Javascript

document.querySelectorAll('[href]')
Ejemplo

Lista de selectores (,)

Coincide con elementos especificados, separados por coma

CSS

div, span {
    color: red;
}

Javascript

document.querySelectorAll('div, span')
Ejemplo

Selector de descendientes ( )

Coincide con el último elemento especificado, que es un descendiente de los elemento anteriores

CSS

div span {
    color: red;
}

Javascript

document.querySelectorAll('div span')
Ejemplo

Selector de hijos (>)

Coincide con el último elemento especificado, que es un hijo (o descendiente directo) de los elementos anteriores

CSS

div > span {
    color: red;
}

Javascript

document.querySelectorAll('div > span')
Ejemplo

Selector de hermanos (~)

Coincide con el ultimo elemento especificado, que es un hermano de los elementos anteriores

CSS

div ~ span {
    color: red;
}

Javascript

document.querySelectorAll('div ~ span')
Ejemplo

Selector de hermanos adyacentes (+)

Coincide con el ultimo elemento especificado, que es un hermano adyacente (o inmediato) de los elementos anteriores

CSS

div + span {
    color: red;
}

Javascript

document.querySelectorAll('div + span')
Ejemplo
